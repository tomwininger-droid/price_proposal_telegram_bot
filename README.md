## איך פותחים
פתחו את הקובץ `index.html` בדאבל-קליק — הוא יפתח בדפדפן הברירת מחדל.

## איך עורכים
לוחצים על כל טקסט במסמך (כותרת, פסקה, שורה בטבלה) ומקלידים ישירות במקום.
הלחיצה מחוץ לטקסט שומרת את השינוי אוטומטית.

- **➕ הוסף פסקה** — מוסיף פסקה לסעיף.
- **➕ הוסף סעיף חדש** — בתחתית המסמך.
- **➕ הוסף שורה** בטבלת המחירים — הסכומים והמע״מ מתעדכנים אוטומטית.
- אפשר לשנות את אחוז המע״מ בשדה הקטן שליד "מע״מ".

## שמירה וטעינה
- **💾 שמור הצעה** — שומר את ההצעה הנוכחית ברשימת ההצעות השמורות (במחשב שלך, בדפדפן).
- **📂 טען הצעה** — פותח רשימה של הצעות שמורות לטעינה או מחיקה.
- **⭐ שמור כתבנית** — שומר את התוכן הנוכחי כתבנית לשימוש עתידי ב"חדש".
- **📄 חדש** — פותח הצעה ריקה מהתבנית האחרונה שנשמרה (או מהתבנית המקורית).

## גיבוי / העברה בין מחשבים
- **⬆️/⬇️ ייצוא / ייבוא גיבוי** — מייצא את כל ההצעות והתבניות לקובץ JSON אחד, וניתן לייבא אותו בחזרה (גם במחשב אחר).

## הורדת PDF
לוחצים על **🖨️ הורד PDF** — נפתח חלון ההדפסה של הדפדפן. בוחרים "Save as PDF" (או מדפסת PDF)
ולוחצים שמירה. הפלט יהיה נקי, ללא סרגל הכלים, באיכות וקטורית חדה.

## חשוב לדעת
התוכן נשמר ב-`localStorage` של הדפדפן — אם מנקים נתוני דפדפן (Clear browsing data) התוכן יימחק.
מומלץ לבצע ייצוא גיבוי מעת לעת כדי לא לאבד הצעות חשובות.
